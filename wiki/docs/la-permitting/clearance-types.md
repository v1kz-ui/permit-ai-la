# Clearance Types

Complete taxonomy of clearances tracked by PermitAI, organized by department.

## LADBS

| Type | Description | Required |
|------|-------------|----------|
| Building Permit | Building permit clearance | Always |
| Electrical Permit | Electrical permit clearance | Always |
| Plumbing Permit | Plumbing permit clearance | Always |
| Mechanical Permit | Mechanical/HVAC permit clearance | Always |
| Grading Permit | Grading permit for hillside properties | Hillside only |

## DCP (City Planning)

| Type | Description | Required |
|------|-------------|----------|
| Zoning Clearance | Zoning conformance verification | Always |
| Coastal Development Permit | Coastal Development Permit | Coastal zone |
| Specific Plan Compliance | Specific Plan area compliance review | Specific plan areas |
| HPOZ Review | Historic Preservation Overlay Zone review | HPOZ areas |

## BOE (Bureau of Engineering)

| Type | Description | Required |
|------|-------------|----------|
| Sewer Connection | Sewer connection/capacity clearance | Always |
| Street Improvement | Street/sidewalk improvement requirements | Conditional |
| Haul Route | Haul route approval for demolition debris | Hillside |

## LAFD (Fire Department)

| Type | Description | Required |
|------|-------------|----------|
| Fire/Life Safety | Fire/life safety plan review | Always |
| Brush Clearance | Brush clearance verification | VHFSZ |
| Fire Flow | Fire flow/hydrant adequacy | VHFSZ |

## LADWP (Water & Power)

| Type | Description | Required |
|------|-------------|----------|
| Water Service | Water service connection clearance | Always |
| Power Service | Electrical service connection clearance | Always |

## LASAN (Sanitation)

| Type | Description | Required |
|------|-------------|----------|
| LID Compliance | Low Impact Development stormwater compliance | Always |
| Green Streets | Green streets requirement | Conditional |

## LAHD (Housing)

| Type | Description | Required |
|------|-------------|----------|
| Rent Stabilization | Rent Stabilization Ordinance clearance | Multi-unit only |

## DOT (Transportation)

| Type | Description | Required |
|------|-------------|----------|
| Driveway Access | Driveway/curb cut access review | Conditional |
| Traffic Study | Traffic impact analysis for major projects | Major projects |

## Cultural Affairs

| Type | Description | Required |
|------|-------------|----------|
| Cultural Resource Review | Cultural/archaeological resource review | Historic properties |
| Public Art Compliance | Public art requirement (projects >$500K) | Conditional |

## Urban Forestry

| Type | Description | Required |
|------|-------------|----------|
| Tree Removal Permit | Protected tree removal/relocation permit | Conditional |
| Tree Replacement | Tree replacement plan compliance | Conditional |

## LA County

| Type | Description | Required |
|------|-------------|----------|
| Geotechnical Review | Geotechnical/soils report review | Geological hazard |
| Flood Zone Review | Flood zone compliance review | Flood zone |
